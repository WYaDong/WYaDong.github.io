相关内容说明

1.ppi = (物理像素宽)与(物理像素高)的平方的和,开方后 除以 (屏幕尺寸); ppi越高,颗粒感越弱,分辨率越高。
2.dpi = (开发尺寸宽)与(开发尺寸高)的平方的和,开方后 除以 (屏幕尺寸)
3.plus系列的PPI是401，理论上苹果应该采用401 / 326 * @2x = @2.46x的素材，这个比例对于开发者而言很难切图，所以Apple为了方便开发者就采用了@3x的素材，然后再缩放到@2.46x的屏幕上，也就是缩放到2.46 / 3 = 82%。实际上Apple选取了一个接近比例的87%，这样算下来，物理分辨率和虚拟分辨率的比例是87%，也就是1080 / 1242 ≈ 0.87， 1920 / 2208 ≈ 0.87。
